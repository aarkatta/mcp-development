DrugBot UI
Next.js 16 • React 19 • TypeScript
  • ChatGPT-style conversational interface
  • useChat() hook manages session state
  • Markdown rendering (react-markdown)
  • ShadCN UI + TailwindCSS v4 styling
  • Framer Motion animations
  • Dark/Light theme toggle
  • Responsive mobile-first design
  • Suggested query chips on start
ü§ñ
MCP Client (Agent)
FastAPI • Gemini SDK • Python 3.12
  • LLM orchestration with Gemini 3 Pro
  • Session management (in-memory dict)
  • MCP protocol client (SSE transport)
  • Agentic tool-calling loop (max 10 turns)
  • System instructions (80+ lines of rules)
  • POST /chat, DELETE /session, GET /health
  • Structured response formatting
  • Tool execution logging
üîß
MCP Server
FastMCP 2.0 • httpx • Python 3.12
  • 12+ FDA tools exposed via MCP protocol
  • SSE transport for bidirectional comms
  • OpenFDA API wrapper (4 endpoints)
  • Smart data filtering (60-80% reduction)
  • Adverse events, labels, recalls, shortages
  • Token-optimized responses for LLM
  • Async HTTP with httpx
  • Optional API key for higher rate limits
